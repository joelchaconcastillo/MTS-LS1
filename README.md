# MTS-LS1

###### Sequential implementation of Multiple Trajectory Search (MS) - L.S (1) for Global Optimization. The full algorithm uses 3 different mechanisms, the source code has only the first one.

***

##### Compile

```sh
$ cd repo
$ make
```
##### Execute

```sh
$ cd repo
$ ./mts-app or make run
```

#### Clean up

```sh
$ make clean
```
